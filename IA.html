Unit6

Creating Agent Dialogues

Create an agent dialogue, using KQML and KIF, between two agents (named Alice and Bob).

Alice is an agent designed to procure stock and Bob is an agent that controls the stock levels for a warehouse. This dialogue should see Alice asking Bob about the available stock of 50 inch televisions, and also querying the number of HDMI slots the televisions have.

Alice asking Bob about the available stock of 50 inch televisions

(KQML-AGENT-SEND :receiver Bob :content (ASK :receiver all-items :content (ITEM-NAME ?item "50 inch television")))

Alice asking Bob about the number of HDMI slots the televisions have

(KQML-AGENT-SEND :receiver Bob :content (ASK :receiver all-items :content (HAS-SLOT ?item HDMI-slots the televisions have))))

NOUN, VERB, PRON : pronoun, ADP : adposition, DET : determiner

The government raised interest rates.

The: DET - government: NOUN - raised: VERB - interest: NOUN - rates: NOUN

The internet gives everyone a voice.

The: DET　- internet: NOUN - gives: VERB - everyone: PRON - a: DET - voice: NOUN

The man saw the dog with the telescope.

The: DET - man: NOUN - saw: VERB - the: DET - dog: NOUN - with: ADP - the: DET - telescope: NOUN

・LLM specialising in medicines

-Enter the name of a disease and the system instantly determines which drugs are effective for that disease. And if there is no effective drug, the AI will create one itself.

-Some medicines are not authorised in some countries, so LLMs need to be created for each country.

-AI producing drugs will reduce research costs. But there will be a certain number of people who will reject AI-generated drugs because they could cause serious side effects and other problems.

・AI chatbots specific to their web services

-The chatbot's answers are accurate because it specialises in that web service. This can answer every detail about all functions of the web service.

-Reduction in the number of employees as they spend less time dealing with customers. Customers will be more satisfied with web services as they can get accurate answers instantly
